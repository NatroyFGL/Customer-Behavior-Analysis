RFM Customer Segmentation Analysis
A end-to-end customer segmentation project using RFM (Recency, Frequency, Monetary) analysis to identify customer behavior patterns and support business decision-making.

Tools & Technologies
Google BigQuery — data storage and SQL querying
SQL — data cleaning, validation, RFM scoring, cohort analysis
Power BI — interactive dashboard and visualization

Dataset
13.3M+ credit/debit card transactions across 1,219 users
Total transaction volume: $5.4M
Tables: orders, order_items, payments, products, reviews

Key Findings
Lost segment is the largest at 26% of customers but contributes only 12.2% of revenue
VIP + Loyal combined drive over 38% of total revenue with ~23% of customers
Big Spender segment has the highest revenue per customer despite mid-range frequency
Cohort retention analysis reveals drop-off patterns useful for re-engagement campaigns
